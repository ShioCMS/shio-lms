# Installation

## Download

```shell
$ git clone https://github.com/ShioCMS/shio-lms.git
$ cd shio-lms
```

## Deploy 

### 1. Runtime

Use NPM and Gradle to execute Shiohara Stock.

```shell
$ ng serve --open
$ ./gradlew bootrun
```

### 2. Build

Use NPM and Gradle to build Shiohara Stock.

```shell
$ ng build --prod
$ ./gradlew build
```
